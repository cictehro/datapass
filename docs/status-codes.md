# Status Codes

| Code | Description                           |
| ---- | ------------------------------------- |
| vf   | Visa Free                             |
| vo   | Visa On Arrival                       |
| ev   | eVisa                                 |
| et   | Electronic Travel Authorization (ETA) |
| vr   | Visa Required                         |

## Notes

* `days` represents maximum permitted stay.
* Empty `days` means not applicable or unknown.
